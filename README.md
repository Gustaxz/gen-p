# GEN P

## How to use

### Requirements

-   Node v20.10.0
-   Yarn v1.22.21
-   Npm v10.2.3

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/Gustaxz/gen-p.git
    ```
2. Install packages
    ```sh
    yarn
    ```
3. Build the project
    ```sh
    yarn build
    ```
4. Install global
    ```sh
    npm i -g .
    ```
5. Run the project
    ```sh
    genp generate <jsonSchemaPath>
    ```
